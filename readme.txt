== Description ==
Create high converting landing pages, thank you pages, reservation pages, business pages, event pages, sales pages, coming soon pages, product pages, lead pages, and just about any kind of page.

== Installation ==
1. Download the plugin

2. Go to your Dashboard and locate Plugins tab. Click on Upload Plugin.

3. Click on Activate Plugin

== Frequently Asked Questions ==
= Is this plugin free?
Yes, the plugin is free to use. However, a basic subscription to MuchPages will only allow you to create one page at a time. Please upgrade to create multiple pages and to unlock other great features.
